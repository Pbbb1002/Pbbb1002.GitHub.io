线性规划

线性规则求解需清晰两部分，目标函数（max， min）和约束条件（s.t.），求解前应转化为标准形式：

目标函数系数 (c)：目标函数是一个线性表达式，由变量的系数组成。目标是最小化目标函数的值。c 是一个包含每个变量对应系数的向量。
不等式约束系数矩阵 A和不等式约束右侧向量 b：不等式约束表示为 A* x ≤ b，其中 A是一个矩阵，每行表示一个约束条件的系数，b 是一个向量，表示对应约束条件的右侧值。
等式约束系数矩阵 Aeq 和等式约束右侧向量 beq：等式约束表示为 Aeq * x = beq，其中 Aeq 是一个矩阵，每行表示一个约束条件的系数，beq 是一个向量，表示对应约束条件的右侧值。
变量的下界lb和ub：变量的下界是一个向量，表示每个变量的最小取值限制。默认情况下，变量没有下界限制，可以通过设置 bounds 参数来指定。

Python高阶技巧

闭包

定义双层嵌套函数，内部函数可以访问外层函数的变量，将内存函数作为外层函数的返回，此内存函数就是闭包函数

优点：无需定义全局变量即可实现通过函数持续访问、修改某个值；闭包使用的变量所用于函数内，难以被错误的调用修改
缺点：由于内部函数持续引用外部函数的值，所以会导致这一部分内存空间不被释放，一直占用内存

数据库和SQL

数据库就是指数据存储的库，作用就是组织数据并存储数据。数据库按照库 — 表 — 数据三个层级进行组织。而SQL语言是用于访问和处理数据库的标准的计算机语言

SQL基础

SQL语言的分类

由于数据库管理系统（数据库软件）功能非常多，不仅仅是存储数据还包含数据的管理、表的管理、库的管理、账户管理、权限管理等等。

所以操作数据库的SQL语言也基于功能可以划分为四类：

· 数据定义：DDL —— 实现库的创建删除、表的创建删除等
· 数据操纵：DML —— 实现新增数据、删除数据、修改数据等
· 数据控制：DCL —— 实现新增用户、删除用户、密码修改、权限管理等
· 数据查询：DQL —— 基于需求查询和计算数据

SQL的语法特征

· SQL 语言大小写不敏感
· SQL 可以单行或者多行书写，最后以” ; “结束
· SQL 只支持单引号
· SQL支持注释：
· 单行注释：–注释内容（–后面一定要有一个空格）
· 单行注释：# 注释内容（# 后面可以不加空格，推荐加上）
· 多行注释：/* 注释内容 */

面向对象

面向对象编程（Object-Oriented Programming，简称OOP）是一种编程范式，它的核心思想是将数据和对数据的操作（行为）封装在一个对象中，对象之间通过消息传递进行交互(设计类，基于类创建对象，由对象做具体的工作)。面向对象编程强调通过封装、继承、多态等特性来组织代码和数据，使得程序更加模块化、可扩展、易维护和可重用。

简单的面向对象演示如下：

类

类是一种抽象数据类型，用于描述具有相同属性和行为的一组对象。类是一种模板或者蓝图，它定义了对象的属性和方法，但并不包含实际数据或实现

类的使用语法：

· class是关键字，表示要定义类了
· 类的属性即定义在类中的变量 (成员变量)
· 类的行为即定义在类中的函数 (成员方法)

文件

操作系统中以文件为单位管理磁盘中的数据，一般来说文件可分为文本文件、视频文件、音频文件、图像文件、可执行文件等多种类别

文件的编码

文本文件通过使用编码技术将内容翻译成0和1被计算机识别，从而存储在硬盘中。编码技术就是翻译的规则集合，记录内容和二进制间相互转换的逻辑，不同的编码将内容翻译成二进制的内容也是不同的，UTF-8是目前全球通用的编码格式。

计算机只能识别0和1，所以只有将内容翻译成0和1才能保存在计算机中，同时也需要编码将计算机保存的0和1反向编译回可以识别的内容

文件的操作

文件的操作包括打开、关闭、读、写等。对文件的基本操作大致分为三个步骤：

1.打开文件
2.读写文件
3.关闭文件

注意：可以只打开和关闭文件，而不进行任何读写

数据容器

定义

一种可以容纳多份数据的数据类型，容纳的数据称之为元素，每一个元素可以是任意类型的数据，如字符串、数据、布尔等。

根据数据容器特点不同（如是否支持重复元素、是否可以修改、是否有序等）分为五类，分别为 ：
列表（list） 元组（tuple） 字符串（str） 集合（set） 字典（dict）

方法

在Python中，如果将函数定义为class（类）的成员，那么函数会被称为方法，方法和函数功能一样，有传入参数，有返回值，只是方法的使用格式不同
