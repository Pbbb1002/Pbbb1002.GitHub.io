json数据格式

json是一种轻量级的、在各个编程语言中流通的数据交互格式，可以按照json指定的格式组织和封装数据，其本质是一个带有特定格式的字符串，负责不同编程语言间数据传递和交互。

Python使用json具有很大优势，因为json是一个单独的字典或者一个内部元素都是字典的列表，所以json可以直接和Python的字典或者列表无缝转换

数据处理

有些时候文件较为复杂，可以逐步简化文件数据
ab173.com：一个懒人网站，可以帮助查看嵌套层级较多的json数据、可以查看色号

步骤:

  1. 使用列表/字典中的replace方法以及序列的切片去掉文件中不合json规范的部分

  2. 通过print函数逐步查看数据是否符合要求，若符合则复制生成信息重新生成json视图，直至文件数据最简

  3. 创建一个空元组用于接受数据

  4. 使用for循环对数据进行遍历，并根据需求将数据存入新的列表中，最后将新列表追加到元组中
