PyCharm常用快捷键

ctrl+alt+s ： 打开软件设置
ctrl + d ：复制当前行代码
shift + alt + 上/下 ：将当前代码向上或向下移动
shift + f6 : 重命名文件
ctrl + a ：全选
ctrl + c\v\x : 复制、粘贴、剪切
ctrl + f ：查找
ctrl + shift + f10 ： 运行当前代码

注释

注释的作用

对程序代码进行解释说明，大大增强代码可读性，不影响代码执行

注释的分类

单行注释：

以# 开头，# 右边所有文字作为注释内容

多行注释 :

以一对三个引号将注释内容引起来

注释的快捷键是Ctrl + /

变量

什么是变量

在程序运行中，储存计算结果或者能表示值的抽象概念

变量的定义格式

变量名 = 变量值 / 变量表达式

数据类型

type语句：用于获取数据的类型

数据类型的转换

注意事项：
整数、浮点数可以转换为字符串；若将字符串转换为数字，则字符串内容必须为数字。
浮点数转换成整数将会把小数部分直接删去（并不是四舍五入），导致精度降低。

标识符

标识符是在编程时所使用的一系列名字，用于给变量、类、方法等命名。

标识符命名规则

1.内容限定 ：

标识符命名中只允许出现英文、中文（不推荐使用）、数字、下划线，且不可以以数字开头。

2.大小写敏感

字母大小写不同，标识符也不同

3.不可以使用关键字

关键字同样对大小写敏感

变量命名规范

简单明了，变量中的英文字母应该全部小写，单词与单词之间用下划线连接

运算符

算术运算符

运算符 描述
+ 两个对象相加
- 得到负数或者一个数减去另一个数
* 两个数相乘或者返回一个被重复若干次的字符串
/ x除以y
% 返回除法的余数
** 返回x的y次幂
// 取整，返回商的整数部分

赋值运算符

运算符 描述 实例
= 简单的赋值运算符 c = a + b
+= 加法赋值运算符 c +=a 等效于 c = c + a
-= 减法赋值运算符 c -=a 等效于 c = c - a
‘*’= 乘法赋值运算符 c ‘*’= a 等效于 c = c * a
/= 除法赋值运算符 c /= a 等效于 c = c / a
%= 取模赋值运算符 c %=a 等效于 c = c % a
//= 取整除赋值运算符 c //=a 等效于 c // c + a

比较运算符

运算符 描述
== 等于
！= 不等于
> 大于
< 小于
>= 大于等于
<= 小于等于

位运算符

实例中 a = 60（二进制 0011 1100），b = 13（二进制 0000 1101）

运算符 描述 实例
& 按位与运算符 a&b 输出结果为 12，二进制解释： 0000 1100
| 按位或运算符 a|b 输出结果为 61，二进制解释： 0011 1101
^ 按位异或运算符 a ^ b 输出结果为 49，二进制解释：0011 0001
~ 按位取反运算符 a~ 输出结果为-61，二进制解释：1100 0011，是一个有符号二进制数的补码形式
<< 左移动运算符 a << 2 输出结果为 240 ，二进制解释： 1111 0000
>> 右移动运算符 a >> 2 输出结果为 15 ，二进制解释： 0000 1111

逻辑运算符

实例中 a = 10，b = 20

运算符 逻辑表达式 描述 实例
and x and y 布尔”与”，如果x为False，x and y 返回 False，否则返回y的计算值 a and b返回 20
or x or y 布尔”或”，如果x为非0，返回 x的值，否则返回y的计算值 a or 20返回 10
not not x 布尔”非”，如果x为True， 返回 False，反之亦然 not(a and b)返回False

成员运算符

运算符 描述
in 如果在指定的序列中找到值则返回True，否则返回False
not in 如果在指定的序列中没有找到值则返回True，否则返回False

字符串拓展

字符串的三种定义方法

单引号定义法可以内含双引号
双引号定义法可以内含单引号
可以使用转移字符（\）来将引号解除效用变成普通字符串

三引号定义法和多行注释写法一样，支持换行操作。用变量接收它则为字符串，不接收则可作为多行注释使用

字符串拼接

如果有多个字符串 /字面量/变量，可以将其拼接成一个字符串，通过+号即可完成，但字符串不能和整数、浮点数等非字符串类型进行拼接

字符串格式化

方法一：使用占位符

与C语言类似，使用占位符%来进行字符串格式化；多个变量占位需要用括号括起来，并按照占位的顺序填入

格式符号 转化
%s 将内容转化为字符串，放入占位位置
%d 将内容转化为整数，放入占位位置
%f 将内容转化为浮点数，放入占位位

可以使用辅助符号来控制数据的宽度和精度
· m， 控制宽度，要求是数字，当设置的宽度小于数据自身时不生效，当宽度大于数据时，将从左开始用空格将宽度补齐
· .n, 控制小数点精度，要求时是数字，会进行小数的四舍五入

方法二：通过f “内容” {变量} 的格式来进行格式化

特点： 不对数据类型作限制，不做精度控制，适合对精度没有要求时快速使用

input函数的使用

与print函数相对，可以使用input函数来简单从键盘获取输入数据
可以使用：input(提示信息)，用以在输入之前显示提示信息
要注意的是，无论键盘输入了什么类型的数据，接收的数据默认为字符串类型

布尔类型

布尔（bool）表示现实生活中的逻辑，即真和假，True表示真，记作1；False表示假，记作0。

布尔类型不仅可以自行定义，也可以通过使用比较运算符进行比较得到结果
